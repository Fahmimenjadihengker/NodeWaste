# NodeWaste

NodeWaste adalah web app edukasi dan gamifikasi pengelolaan sampah. Struktur project dibuat sebagai monorepo sederhana dengan dua folder aplikasi utama.

## Struktur Utama

```txt
NodeWaste/
|-- frontend/  # React + Vite + Tailwind CSS
|-- backend/   # Express API
```

## Menjalankan Project

Frontend:

```bash
cd frontend
npm install
npm run dev
```

Backend:

```bash
cd backend
npm install
npm run dev
```
